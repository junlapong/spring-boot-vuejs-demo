# Spring Boot Demo

## Frontend

```
cd web

# develop
yarn serve

# build
yarn build
```

## Backend

```
# develop
./mvnw spring-boot:run

# test
./mvnw test

# build
./mvnw clean package -Dmaven.test.skip
```
